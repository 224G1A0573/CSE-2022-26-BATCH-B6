# Adaptive Technology for Autism Support

## Project Overview
Autism Spectrum Disorder (ASD) affects communication, emotional regulation, attention, and social interaction in children. Traditional therapy and assistive methods often lack personalization, real-time adaptability, and continuous monitoring, which limits their effectiveness across diverse behavioral needs.

This project proposes an AI-assisted cross-platform application designed to support children aged 6–12 years with Autism Spectrum Disorder. The system uses emotion recognition, eye tracking, and speech-based cues to understand a child’s emotional state and attention level in real time. Based on this analysis, the application provides adaptive feedback through a virtual avatar and interactive therapeutic games.

## Objectives
- Develop an adaptive support system for children with ASD
- Detect emotional state and attention using facial expressions, eye tracking, and speech cues
- Provide personalized feedback through virtual avatars and therapeutic games
- Improve emotional regulation, focus, and basic social skills
- Enable caregivers and therapists to monitor progress

## Technologies Used
- Dart
- Flutter
- Python
- Firebase
- Android Studio

## Key Features
- Real-time facial emotion recognition
- Webcam-based eye tracking
- Speech emotion detection
- Interactive therapeutic games
- Caregiver and therapist progress monitoring
