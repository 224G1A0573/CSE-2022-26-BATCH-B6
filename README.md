# Detection of Pneumonia Using Neural Networks

## Project Overview
Pneumonia is a serious respiratory disease that requires early and accurate diagnosis to reduce mortality rates. Traditional diagnosis using chest X-ray images depends on expert radiologists and can be time-consuming and subjective.

This project proposes a **deep learningâ€“based system** to automatically detect pneumonia from **chest X-ray images** using pretrained **Convolutional Neural Network (CNN)** models. The system aims to assist healthcare professionals by providing a fast and reliable diagnostic support tool.

---

## Objectives
- To detect pneumonia from chest X-ray images using neural networks  
- To compare different pretrained CNN models for performance  
- To improve accuracy in pneumonia classification  
- To support early diagnosis through automated image analysis  

---

## Technologies Used
- **Programming Language:** Python  
- **Deep Learning Framework:** TensorFlow / Keras  
- **Models:** VGG16, ResNet50, InceptionV3  
- **Libraries:** NumPy, Pandas, OpenCV, Matplotlib  
- **Development Environment:** Jupyter Notebook  

---

## Dataset Description
- **Source:** Radiological Society of North America (RSNA)  
- **Data Type:** Chest X-ray images (Anterior-Posterior view)  
- **Classes:** Normal, Pneumonia  
- **Dataset Split:** Training, Validation, Testing
