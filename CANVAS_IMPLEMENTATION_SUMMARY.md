## 🎮 HOW IT WORKS:

### **Child Side:**
```
Dashboard → "Draw Together 🎨" → Select Therapist → Start Drawing!
                                                           ↓
                                                   Emotions = Colors!
```

### **Therapist Side:**
```
Children Tab → Select Child → "Draw" Button → Canvas Opens
                                                      ↓
                                              Draw Together in Real-Time!
```
